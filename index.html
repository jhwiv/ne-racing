Advice Engine Methodology

The Advice Engine computes a composite score (0–100) for each horse in each race, then assigns a letter grade and generates a one-line rationale. The five factors and their weights are:

FactorWeightMethod
Speed35%Average of last 3 Beyer/speed figs, normalized 0–100. +5 bonus if most recent is highest.
Class20%Map race type to numeric scale (G1=100…MCL=20). Dropping in class = bonus; rising = penalty.
Pace20%Identify running styles. Lone speed gets +15. Hot pace (3+ front-runners) gives closers +10.
Trainer/Jockey15%Win % input: >20% = 90pts, 15–20% = 70, 10–15% = 50, <10% = 30.
Post + Bias10%Cross-reference today's bias log entry (rail, running style adv.) with PP and style.

Letter grades: A+ (90–100), A (80–89), B+ (73–79), B (65–72), C (50–64), D (<50). Confidence level reflects score separation between top two horses.
